# Form to Multi-Channel Notifications

[![n8n](https://img.shields.io/badge/n8n-workflow-orange)](https://n8n.io/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A production-ready n8n workflow automation that captures form submissions and instantly distributes them across multiple notification channels: Email, Telegram, and Google Sheets.

## 🎯 Overview

This workflow automates lead capture and notification distribution for businesses receiving form submissions. When a form is submitted, the data is processed and simultaneously sent to:

- **Gmail** - HTML-formatted email notifications
- **Telegram** - Instant messaging alerts
- **Google Sheets** - Structured data storage with status tracking

Perfect for marketing teams, sales departments, and customer support operations that need real-time lead notifications.

## ✨ Features

- **Multi-Channel Delivery** - Simultaneous notifications across email, messaging, and spreadsheets
- **Real-Time Processing** - Instant webhook trigger for zero-delay notifications
- **Data Normalization** - Structured field mapping with timestamps and source tracking
- **Rich Formatting** - HTML email templates and formatted Telegram messages
- **Lead Management** - Automatic status assignment and Google Sheets integration
- **Scalable Architecture** - Parallel node execution for optimal performance

## 🛠️ Technology Stack

- **Platform**: n8n (v1.x+)
- **Trigger**: Webhook (POST endpoint)
- **Integrations**:
  - Gmail API (OAuth2)
  - Telegram Bot API
  - Google Sheets API (OAuth2)

## 📋 Prerequisites

Before deploying this workflow, ensure you have:

1. **n8n Instance** (self-hosted or cloud)
2. **Gmail Account** with OAuth2 credentials configured
3. **Telegram Bot Token** from [@BotFather](https://t.me/botfather)
4. **Google Sheets** with appropriate API access
5. **Webhook URL** (automatically generated by n8n)

## 🚀 Installation

### Step 1: Import Workflow

1. Download `Form-to-Multi-Channel-Notifications.json`
2. Open your n8n instance
3. Navigate to **Workflows** → **Import from File**
4. Select the downloaded JSON file
5. Click **Import**

### Step 2: Configure Credentials

#### Gmail Configuration
1. Go to **Credentials** → **New Credential**
2. Select **Gmail OAuth2 API**
3. Follow Google OAuth2 setup instructions
4. Authorize access to your Gmail account
5. Update the workflow's Gmail node with your credentials

#### Telegram Configuration
1. Create a bot via [@BotFather](https://t.me/botfather)
2. Copy the bot token
3. Get your Chat ID using [@userinfobot](https://t.me/userinfobot)
4. Add **Telegram API** credentials in n8n
5. Update the workflow with your Chat ID

#### Google Sheets Configuration
1. Create a new Google Sheet
2. Set up column headers: `Name`, `Email`, `Phone`, `Company`, `Service`, `Timestamp`, `Source`, `Status`
3. Configure **Google Sheets OAuth2 API** credentials
4. Update the workflow with your spreadsheet ID

### Step 3: Customize Settings

Edit the workflow nodes to match your requirements:

- **Webhook Path**: Change `multi-channel-form` to your preferred endpoint
- **Email Recipient**: Update `s23500515@gmail.com` to your email
- **Telegram Chat ID**: Replace `5094673062` with your chat ID
- **Google Sheet ID**: Update with your spreadsheet ID

### Step 4: Activate Workflow

1. Click **Save** to preserve changes
2. Toggle **Active** to enable the workflow
3. Copy the webhook URL from the Webhook node
4. Test with a POST request

## 📡 API Usage

### Webhook Endpoint

**Method**: `POST`  
**URL**: `https://your-n8n-instance.com/webhook/multi-channel-form`  
**Content-Type**: `application/json`

### Request Body

```json
{
  "name": "John Smith",
  "email": "john.smith@example.com",
  "phone": "+1234567890",
  "company": "Tech Innovations Inc",
  "service": "Automation Consulting"
}
```

### Response

```json
{
  "success": true,
  "message": "Form submitted successfully"
}
```

### Example with cURL

```bash
curl -X POST https://your-n8n-instance.com/webhook/multi-channel-form \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Smith",
    "email": "john.smith@example.com",
    "phone": "+1234567890",
    "company": "Tech Innovations Inc",
    "service": "Automation Consulting"
  }'
```

## 🏗️ Workflow Architecture

```
┌─────────────┐
│   Webhook   │ ──── Receives form submission
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Edit Fields │ ──── Normalizes data & adds metadata
└──────┬──────┘
       │
       ├──────────────────┬──────────────────┐
       │                  │                  │
       ▼                  ▼                  ▼
┌────────────┐    ┌─────────────┐   ┌────────────┐
│   Gmail    │    │  Telegram   │   │  Sheets    │
│  (Email)   │    │  (Message)  │   │  (Storage) │
└────────────┘    └─────────────┘   └────────────┘
```

### Node Descriptions

| Node | Type | Purpose |
|------|------|---------|
| **Webhook** | Trigger | Receives POST requests from forms |
| **Edit Fields** | Set | Extracts and structures form data |
| **Send a message** | Gmail | Sends HTML email notifications |
| **Send a text message** | Telegram | Delivers instant Telegram alerts |
| **Append or update row** | Google Sheets | Stores lead data with status tracking |

## 🎨 Customization Guide

### Modify Email Template

Edit the `Send a message` (Gmail) node:

```html
<!DOCTYPE html>
<html>
<body style="font-family: Arial; padding: 20px;">
  <h2 style="color: #0066cc;">🔔 New Form Submission</h2>
  <table style="border-collapse: collapse; width: 100%;">
    <tr style="border-bottom: 1px solid #eee;">
      <td style="padding: 8px;"><b>👤 Name:</b></td>
      <td style="padding: 8px;">{{ $json.name }}</td>
    </tr>
    <!-- Add more fields as needed -->
  </table>
</body>
</html>
```

### Modify Telegram Message

Edit the `Send a text message` (Telegram) node:

```
*New Form Submission!*

*Name:* {{ $json.name }}
*Email:* {{ $json.email }}
*Phone:* {{ $json.phone }}
*Company:* {{ $json.company }}
*Time:* {{ $json.timestamp }}

_Respond within 5 minutes!_
```

### Add Additional Fields

1. Update the **Edit Fields** node to capture new data
2. Modify notification templates in Gmail and Telegram nodes
3. Add corresponding columns in Google Sheets
4. Update the **Append or update row** node mapping

## 🐛 Troubleshooting

### Webhook Not Triggering
- Verify the workflow is **Active**
- Check webhook URL is correct
- Ensure request Content-Type is `application/json`

### Gmail Not Sending
- Confirm OAuth2 credentials are valid
- Check Gmail API is enabled in Google Cloud Console
- Verify recipient email address is correct

### Telegram Messages Not Delivered
- Validate bot token with [@BotFather](https://t.me/botfather)
- Ensure Chat ID is correct
- Verify you've started a conversation with the bot

### Google Sheets Not Updating
- Check OAuth2 credentials have write permissions
- Verify spreadsheet ID matches your sheet
- Ensure column headers match exactly
